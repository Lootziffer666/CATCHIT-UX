# CatchIt Glossary (Tight)

- **Anchor**: The stable orientation reference that reduces decision stress.
- **Residue**: Leftover cognitive friction/confusion after an interaction step.
- **Chain**: The day/travel sequence context; decisions are chain-aware, not isolated.
- **Main State Family**: Primary operational states representing current truth.
- **Side Swipe**: Fast, today-relevant modulation surface.
- **Backstage**: Durable structural settings and rare-change controls.
- **Quick Change**: Minimal-friction adjustment flow for immediate travel context.
- **Core Gold**: Architecturally central gold references.
- **Extended Gold**: Supporting gold references that complement Core Gold.
