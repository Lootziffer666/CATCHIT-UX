# Gold Set Guidance

## What belongs here
Gold-level reference files used as architecture and quality anchors.

## Authority
Gold reference authority, immediately below canonical rule folders.

## Core vs Extended
- **Core Gold** (`12A_*`): product-architecturally central.
- **Extended Gold** (`12B_*`): supporting gold-level reference.

## Use
Validate proposed changes against gold references before implementation.

## Do not
Do not treat Extended Gold as replacement for Core Gold when conflicts appear.
