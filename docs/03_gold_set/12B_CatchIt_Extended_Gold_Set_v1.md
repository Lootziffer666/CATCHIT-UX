## Working rule
This file identifies the gold-level supporting references that extend CatchIt’s practical coverage and trust posture without replacing the screens that carry its main grammar.

## 1. What Extended Gold Means

An Extended Gold screen is still gold-level.
It is not a weak or optional leftover.

It belongs to Extended Gold because it is:
- supporting rather than grammar-defining
- highly useful, but not the first screen used to explain CatchIt
- complementary to the core state and flow system
- important for practical breadth, trust, and robustness

Extended Gold is not “nice to have.”
It is “gold-level, but not foundational in the same way as Core Gold.”

---

## 2. Extended Gold Set v1

The following seven screens belong to CatchIt Extended Gold Set v1.

### 1. Wizard / Einstieg
Reason:
- important as the first guided-understanding step
- supports planning entry
- less product-defining than Kettenfrage, but still valuable as a strong reference

### 2. Wizard / Bewegungsstil / Tageslage
Reason:
- important for preference-shaped and today-shaped movement logic
- strengthens everyday realism
- supports CatchIt’s frictionless but humane planning behavior

### 3. Go Mode / In Vehicle
Reason:
- important execution support state
- useful reference for holding the next threshold while already in motion
- less iconic than Walk to Stop, but still important

### 4. Neutral Label vs User Label
Reason:
- makes public-safe vs private meaning visible
- important for trust, dignity, and privacy posture

### 5. Lockscreen-safe Notification
Reason:
- important reference for public-safe glanceable behavior
- strengthens real-world usability under exposed contexts

### 6. Voice-safe Assist Screen
Reason:
- important for socially robust spoken support
- strengthens CatchIt’s non-intrusive assistance behavior

### 7. Accessibility Variant / Large Text + Reduced Motion
Reason:
- essential for robust accessibility adaptation
- shows that accessibility is a functional variant, not a separate visual product

---

## 3. Recommended Extended Gold Order

Use the following order when presenting or reviewing the Extended Gold Set:

1. Wizard / Einstieg
2. Wizard / Bewegungsstil / Tageslage
3. Go Mode / In Vehicle
4. Neutral Label vs User Label
5. Lockscreen-safe Notification
6. Voice-safe Assist Screen
7. Accessibility Variant / Large Text + Reduced Motion

This order is recommended because it moves from:
- supportive planning entry
- to in-transit extension
- to trust/privacy/accessibility posture

---

## 4. Extended Gold Redlines

### Redline 1 — Extended must not import foreign product types
Do not let these screens drift into:
- onboarding products
- settings administration
- tracking dashboards
- notification centers
- accessibility side-apps

### Redline 2 — Trust and privacy remain product features
Neutral labels, lockscreen-safe wording, and voice-safe phrasing are not secondary polish.
They are gold-level trust behaviors.

### Redline 3 — Accessibility remains family-coherent
Large text and reduced motion must preserve CatchIt’s system meaning and family identity.

### Redline 4 — Wizard entry screens stay everyday, not administrative
Wizard / Einstieg and Bewegungsstil must remain guided understanding, not setup/configuration.

### Redline 5 — In Vehicle stays threshold-led
Go Mode / In Vehicle must not drift into tracking dashboard or transit status page behavior.

---

## 5. How to Use Extended Gold

Use Extended Gold:
- to extend the custom GPT’s reference quality beyond the core screens
- to stabilize trust/privacy/accessibility behavior
- to keep supporting flows aligned with the main grammar
- to review non-core but still product-important screens

When a new supporting screen is proposed, ask:
- does it extend the core grammar without replacing it?
- does it stay family-coherent?
- is it stronger than the current Extended Gold example?

Do not treat Extended Gold as lower-quality reference.
Treat it as gold-level support reference.

---

## 6. Compact Summary

CatchIt Extended Gold Set v1 contains the gold-level supporting references for:
- planning entry
- movement-reality questioning
- in-vehicle threshold support
- privacy/public-safe behavior
- voice-safe assistance
- accessibility adaptation

These screens are gold-level and important.
They simply do not carry the main grammar in the same way as Core Gold.

## Final Rule
Extended Gold must support and strengthen Core Gold, never compete with or redefine it.
