# Screen_05_Gold_Settings_Backstage_Home

## Purpose
Defines the gold-status version of the CatchIt Settings / Backstage Home screen.

## Status
Gold-status supporting screen for CatchIt Layer 2.

## 1. CatchIt interpretation

This is not the main CatchIt world and not a generic settings graveyard.

It is the calm backstage layer for durable, rarely changed, structurally important foundations.

The point is not showing all options.
The point is making long-term foundations findable, limited, and dignified.

## 2. State type

**System Layer / Backstage Home**

- Screen role: entry into durable backstage structure
- Primary anchor: **Selection Anchor**
- User role: find and manage rarely changed foundations
- CatchIt role: create structural clarity without turning life into administration
- Density: **low-to-medium**
- Tone: calm, functional, dignified, non-technocratic

## 3. Must stay the same

- clearly secondary to Main and Side Swipe
- no dashboard
- no card swarm
- no control-panel aesthetics
- no feature-startpage behavior
- categories remain few and clear
- still visibly CatchIt, but quieter than Main
- no deep nesting on first level

## 4. Must change

Compared to Side Swipe, this becomes:
- less about today
- more about durable structure
- less about modifiers
- more about foundations

First level should only show major backstage areas:
- **Zuhause & Familie**
- **Sicherheit**
- **Vibration & Sound**
- later optional:
  - **Integrationen**
  - **Datenschutz / Konto**

## 5. Drift risks

- Android-settings drift
- control-panel drift
- too many categories
- too many icons and status markers
- feature-list behavior
- too much technical language
- losing CatchIt tone completely

## 6. Recommended screen structure

### Header
Quiet, secondary.

Good title:
- **Einstellungen**

### Orientation line
Optional, short:
- *Selten gebraucht, aber wichtig*

### Area group
A few large, calm entries only.

Examples:
- **Zuhause & Familie** — *feste Orte und Bezugspersonen*
- **Sicherheit** — *Notfallkontakt und Hilfen*
- **Vibration & Sound** — *Rückmeldungen und Modi*

### Layout
- calm large rows or grouped surfaces
- no mini-cells
- no admin-table logic
- no heavy footer

## 7. Prompt for generation tool

Design the CatchIt **Settings / Backstage Home** screen as a calm secondary system layer for long-term structure and rarely changed foundational information, not as a generic Android settings page, admin dashboard, or control panel.

Semantic meaning:
this is the backstage layer for durable personal structure, safety, and sensory settings.

Requirements:
- keep the screen clearly secondary to Main and Side Swipe
- show only a few high-level backstage areas
- avoid settings-app clutter
- avoid dashboard cards
- avoid technical overload
- avoid feature-list behavior
