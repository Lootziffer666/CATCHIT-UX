# CatchIt Design – Teil 3 (bereinigt)

Dieses ZIP ist die bereinigte, gesplittete Fassung von **CatchIt_Design_Teil_3.md**.

## Befund
Teil 3 war inhaltlich vor allem ein:
- Figma Screen Spec
- Mockup-/Präsentationspaket
- Prototype-/Gesten- und Transition-Teil
- Annotierungsleitfaden

Gleichzeitig enthielt die Datei noch ältere oder inkonsistente Stellen wie:
- alte Screennamen (`Stable`, `Bald Relevant`, `Fragile`)
- Vorstände vor der 5-State-Logik
- unklare Trennung zwischen Screen-Set, Präsentationsreihenfolge und Figma-Arbeitslogik

## Ziel dieser Fassung
Teil 3 ist jetzt als **Mockup-, Prototype- und Präsentationssystem** neu geordnet.

2. `02_Core_Screens_und_Mockup_Logik.md`
