# maybe_deprecated — Preserved but Unsafe (Non-Canonical)

## Hard rule
Everything in this folder is **non-canonical**.

## What this folder is for
This is a holding area for:
- traceability,
- unresolved historical material,
- temporary preservation during cleanup.

## What this folder is *not*
- Not a source of truth.
- Not a safe baseline for new work.
- Not a place to copy from by default.

## Required handling
- Do **not** extend files here as if they are active specs.
- Do **not** reference files here as normative in new docs.
- Do **not** let content here silently override canonical docs.

If a file here becomes relevant again, it must be handled deliberately:
1. review against current canonical + gold rules,
2. move to the correct active docs area,
3. relabel/rewrite for current authority,
4. leave a trace note of what changed and why.

Until those steps are completed, treat every file here as historical context only.

## Naming note
The current folder name is intentionally conservative but ambiguous. A future low-risk rename to something explicit (for example `historical_holding_noncanonical/`) is recommended.
