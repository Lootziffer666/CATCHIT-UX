# CatchIt UX Repository

CatchIt is a **decision-relief and reassurance system for public mobility**: state-based, trust-centered, chain-aware, and preference-shaped.

It is **not** a generic transit app, route list, map-first UI, dashboard, or settings-heavy planner.

## What this repository contains
- Canonical product language and UX/system rules (`docs/01_philosophy`, `docs/02_system`)
- Gold references (`docs/03_gold_set`)
- Screen and layer specs (`docs/04_screens`, `docs/05_layers`)
- Motion/branding specs (`docs/06_motion_brand`)
- Archived prototypes and exports (`docs/07_prototypes_archive`)
- Working notes and intermediate docs (`docs/08_working_notes`)
